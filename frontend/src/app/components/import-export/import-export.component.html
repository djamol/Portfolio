Import Data

Upload a CSV file to add/update your investment data. Existing records will be updated, new records will be added.

CSV Format Requirements:

  • Headers must include: ID, Website/App Name, Investment Type, Sub Type Name, Sub Type Category, Amount, Investment Date
  • Date format should be YYYY-MM-DD
  • Amount should be numeric values
  • Records with matching Website/App Name, Investment Type, Sub Type Name, and Sub Type Category will be updated
  • New records will be added to your portfolio
